Добавление фигуры: клик по кнопке фигуры, затем клик на рабочей области. Esc обнуляет клик по кнопке, пока фигура не брошена на рабочую область.

Удаление фигуры: выбрать фигуру (клик по фигуре), затем Delete.

Перемещение фигуры: перетаскивать мышью с нажатой левой кнопкой.

Изменение цвета фигуры: выбрать фигуру; на панели инструментов выбрать цвет. Левый клик по палитре задает цвет. Правый клик по палитре - выбор другого  цвета.

Вращение прямоугольника: выбрать прямоугольник, вращать колесом мыши.

Изменить размеры прямоугольника: выбрать, затем менять размер мышью с нажатой правой кнопкой.

Изменить положение изломов линии: выбрать линию, затем перетаскивать мышью с нажатой правой кнопкой.

Изменить толщину линии: выбрать линию, задать толщину в поле ввода "толщина" цифрами или стрелками "вверх", "вниз".

Перемещение рабочей области: мышкой при зажатой левой кнопке. При выходе курсора за пределы видимой части рабочей области перемещение прекращается. Размеры рабочей области ограничены.
